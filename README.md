# waterloo-trading-comp

Fair value calculators for the live trading games at WQC's Inaugural Waterloo Trading Competition.

## Calculators

### ETF Trading Frenzy (`etf_calculator.py`)

Tracks estimated bot positions (noise bots and event bots) for securities E, T, and F to compute settlement values and the ETF fair value.

- **Settlement formula:** `Settlement(X) = 500 + (NB(X) + EB(X)) / 100`
- **ETF formula:** `ETF = (E + 2T + 3F) / 6`
- Supports incremental position updates, arbitrage checks between the synthetic ETF and market price, and trading signals (buy/sell singles vs ETF).

### Triplet of Pairs (`triplet_calculator.py`)

Bayesian inference calculator for the Triplet of Pairs game. Three events (A, B, C) are each generated by drawing 5 faces from {1,...,10} to form a d5 die, then rolling it 9 times and summing the results. Contracts settle on pair sums: AB = A+B, BC = B+C, AC = A+C.

- Updates a Bayesian posterior over all possible dice as rolls are observed
- Computes expected event totals, face probability distributions, and posterior entropy
- Infers unobserved events from market prices on the pair contracts
- Outputs fair contract prices and trading signals with edge estimates

### Octomarket (`octomarket_calculator.py`)

Computes fair values for all 8 contracts in the Octomarket game, where Teams A and B each roll a d20 ten times.

| Contract | Settles on |
|----------|------------|
| WIN A / WIN B | Which team's sum is higher |
| ODD A / ODD B | Which team has more odd rolls |
| DIV A / DIV B | Whose product has more divisors |
| SPREAD S / SPREAD L | Whether the absolute sum difference exceeds 5.5 / 20.5 |

- WIN and SPREAD use normal approximations; ODD uses exact binomial enumeration; DIV uses Monte Carlo simulation
- Enter rolls in pairs (`ab 14 7`) as they're revealed each minute and compare fair values to exchange quotes

## Usage

Run any calculator directly:

```bash
python3 etf_calculator.py
python3 triplet_calculator.py
python3 octomarket_calculator.py
```

Each calculator runs an interactive CLI — type `help` inside for available commands.
